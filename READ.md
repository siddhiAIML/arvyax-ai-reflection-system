# AI Reflection Understanding System

## Overview
This project builds an AI system that understands human emotional reflections and guides users toward better mental states.

## What it does
- Predicts emotional state
- Predicts intensity
- Suggests what to do
- Suggests when to do it
- Detects uncertainty

## Output
The system generates a file called:
predictions.csv

It contains:
- predicted_state
- predicted_intensity
- confidence
- uncertain_flag
- what_to_do
- when_to_do

## How to run
Run this command:

python understanding.py
